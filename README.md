This is a frontend e-commerce website developed using HTML, CSS, and JavaScript.
The application allows users to browse fresh vegetables and daily-use products, add items to the shopping cart, and view the total price dynamically.
The project focuses on implementing client-side logic, DOM manipulation, and responsive UI design.

Features:->

Product listing for fresh vegetables and daily-use items

Add to cart functionality

Dynamic cart updates

Real-time price calculation using JavaScript

Responsive design for different screen sizes
